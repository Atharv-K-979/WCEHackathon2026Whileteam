# VESSEL

## Problem Statement

Develop a simple solution to protect people from misinformation, privacy leaks, and unsafe AI content, ensuring safe and responsible use of AI.

---

## Solution

VESSEL is a Chrome extension with three defensive layers:

- **Paste Redaction**  
  Blocks sensitive data before it leaves the clipboard.

- **Prompt Injection Defense**  
  Stops malicious web pages from hijacking AI assistants.

- **Secure Specification Assistant**  
  Uses a local ML model to catch missing security requirements in Jira and Notion tickets.
