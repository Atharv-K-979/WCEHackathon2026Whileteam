# VESSEL Web - Landing Page

This directory contains the source code for the VESSEL Landing Page and Dashboard.

## 🚀 Quick Start

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Start Development Server**
   ```bash
   npm run dev
   ```

3. **Build for Production**
   ```bash
   npm run build
   ```

## 🛠️ Tech Stack
- **Framework**: React
- **Bundler**: Vite
- **Styling**: Vanilla CSS (Premium Dark Theme)

For information about the core security logic, please refer to the [Extension documentation](../Extension/README.md) or the [Root README](../readme.md).
